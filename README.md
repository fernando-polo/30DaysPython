# 30DaysPython 🖥️

Personal progress through the 30 Days of Python challenge. Covers Python fundamentals, data structures, functional programming, OOP, and more — one day at a time.

## 📌 Description

This repository tracks my solutions and exercises from the [30 Days of Python](https://github.com/Asabeneh/30-Days-Of-Python) challenge by Asabeneh Yetayeh.

Each folder corresponds to a day of the challenge, with hands-on exercises designed to progressively build Python skills from the ground up — from basic syntax and data types all the way to modules, file handling, and object-oriented programming.

Both `.py` scripts and Jupyter Notebooks are used throughout the challenge.

## 📦 Tech Stack

- **Language:** Python 3
- **Environment:** Jupyter Notebook, Python scripts
- **Tools:** Git, VS Code

## 📋 Topics Covered

- **Fundamentals:** Data types, variables, arithmetic and comparison operators
- **Strings:** Formatting, methods, slicing
- **Data structures:** Lists, tuples, sets, and dictionaries
- **Control flow:** Conditionals and loops
- **Functions:** Parameters, return values, scope, and lambda functions
- **Functional programming:** `map()`, `filter()`, `reduce()`, list comprehensions
- **Modules:** Importing built-in and custom modules
- **OOP:** Classes, objects, attributes, and methods
- **Error handling:** `try/except` and custom exceptions
- **File handling:** Reading and writing files
- **Utilities:** Regular expressions, date/time, and basic statistics

## 🚀 About the Challenge

The 30 Days of Python challenge is a structured, community-driven program designed to take learners from zero to a solid Python foundation in 30 days. Each day introduces new concepts and ends with practical exercises to reinforce learning.

- How to work with all major built-in data structures

- Applying functional programming patterns in Python

- Handling errors gracefully and working with files
- Using Git to track daily progress consistently
